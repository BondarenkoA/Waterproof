# Waterproof
 
Индикация:
1. зеленый - дежурный режим, питание от сети
2. зеленый мигает - питание от батареи
3. Красный - краны закрыты
4. Красный мигает - срабатывание датчика(краны закрываются или уже закрыты)
5. Мигание поочередно желтый-зеленый - краны открыты принудительно, автоматика отключена
5. Мигание поочередно желтый-красный - краны закрыты принудительно, автоматика отключена
 
кнопки:
1. открыть принудительно
2. закрыть принудительно
